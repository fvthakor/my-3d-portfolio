# Fulaji Thakor — Portfolio

Personal portfolio of **Fulaji Thakor**, Lead Full Stack Engineer & VoIP Specialist with 7+ years building scalable real-time systems, VoIP SaaS platforms, and full-stack web applications.

🔗 **Live:** [fvthakor.in](https://fvthakor.in)

## Tech Stack

React · TypeScript · Vite · Three.js · GSAP · Node.js

## Features

- Interactive 3D avatar with mouse tracking
- Animated career timeline
- Tech stack showcase
- Contact & social links
- Google Analytics 4

## Run Locally

```bash
npm install
npm run dev
```
